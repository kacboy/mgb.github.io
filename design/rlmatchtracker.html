RL Match Tracker

Android App

RL Match Tracker is an Android app developed with a partner to provide Rocket League players with game-to-game data tracking, addressing the gap left by platforms like Tracker Network that only show lifetime stats. Created using Android Studio and an SQL database, the app includes features like automated photo-based match data input, performance graphs, and customizable themes. It was available publicly on the Play Store for a time, but I didn't maintain the app to keep up with Google's developer policy updates.
